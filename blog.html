Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents a webpage as a structured tree of objects, allowing dynamic manipulation of its content, structure, and style.

How do you select on element from the DOM?

I can select an element from the DOM using document.querySelector('selector') or document.getElementById('id').

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a parent element handles events for its child elements, improving efficiency by reducing the number of event listeners.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute() or element.attribute = value, and modify styles by accessing element.style.property = value.
